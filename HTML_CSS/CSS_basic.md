## 색상

### 1. 색상 이름
```
    'black'과 'white'처럼 이름을 적어주는 방식이다. 17가지의 표준색과
    123개 정도의 색이 있다
    기본 색상 이름 : aqua, black, blue, fuchsia, gray, green, lime, maroon,
    navy, olive, orange, purple, red, silver, teal, white, yellow
```
### 2. RGB
```
    앞서 본 색상 이름 방식으로는 다양한 색상을 나타내기 힘들기 때문에, 다양한
    색상을 코드로 표현할 수 있다.

    그 중 하나가 rgb 방식이다. rgb는 각각 빛의 삼원색인 Red, Green, Blue를
    나타내며, 다음과 같이 작성한다.
    > rgb(255,10,0)

    rgb를 먼저 쓰고 괄호를 열고서 각 r,g,b에 해당하는 숫자를 입력해 준다.
    숫자 값은 0~255까지이며, 숫자가 높을 수록 해당 색의 양이 많아집니다. 또한
    전체 색상의 양이 많아질 수록 색은 밝아진다. (0,0,0)은 검은색이며
    (255,255,255)는 흰색이다. 위의 색상 값 (255,10,0)은 Red가 255이고 다른
    Green이 10, Blue가 0 이니 거의 빨간색일거라 예측 가능할 수 있다.
```
### 3. HEX
```
    HEX 방법은 색상을 16진수로 표현한 방법이다. 다음과 같이 작성한다.

    #ffcc00
```


## 스타일을 적용하는 몇가지 방법
```
    1. Link 요소로 연결 : 가장 권장되는 방법(common과 main을 둘다 연결 해준다)
        -<link href="common.css" rel="stylesheet" type="text/css">
        -<link href="main.css" rel="stylesheet" type="text/css">
    
    2. style 요소로 적용
    ->HTML 문서와 CSS 문서를 분리하지 않고 넣는 방법이다. 권장하는 방법은 
    아니지만 해당 스타일을 오로지 해당 페이지만 쓴다거나, 임시적으로 테스트를 
    위한 페이지를 만들 때 주로 사용한다.
    ex) <style type="text/css"></style>

    3. style 속성을 통해 적용 : 해당 요소만 스타일을 줄때 사용
        - <h1 style="color:#fff;font-size:2em;">Hello world</h1>
```

# CSS 선택자
```
    말 그대로 선택의 의미를 가지고 있다. 선택자를 통해서 특정 요소들을 선택하여 스타일을 적용할 수 있게 된다.

    1-1 Rule Set
        - HTML 페이지 안의 특정 요소들을 어떻게 렌더링 할 것인지 브라우저에게 알려주는 CSS 문장이다
    
    1-2 선택자 : p{ color: red; padding: 5px }
    위 문장에서 Rule Set 제일 앞 부분의 선택자 요소를 볼 수 있다
    왼쪽 중괄호가 나오기 전의 부분 모두가 선택자 이다
        - @ 키워드로 시작하는 문장은 Rule Set이 아닌 At-Rule이므로 중괄호 전 부분은 선택자가 아니다.
```

# 선택자의 종류
```
    1. 전체 선택자
        : HTML 페이지 내부의 모든 요소에 같은 CSS 속성을 적용한다.
         - margin, padding의 기본값을 정해둘 때 사용
         - 그러나 페이지의 로딩 속도가 느려질 수 있다.
    
    2. 태그 선택자
        : HTML 요소를 직접 지칭하는 가장 간단한 선택자

    3. 클래스 선택자
        : 주어진 값을 class 속성 값으로 가진 HTML 요소를 찾아 선택한다
        (div.class2 { background: darkgreen; color: yellowgreen; } 같이 class 앞에 태그가 있는경우 특정 태그에 한함)
            클래스 선택자를 사용하기 전 생각해야할 부분
                1. class요소 대신 사용할 수 있는 HTML 태그가 있는지 확인
                - heading 처리한 글씨처럼 보이기 위해 <h1> 대신 class 선택자 사용 -> 스타일 시트를 적용하지 못하는 경우가 생길 수 있음

                2. DOM 트리 상단에 사ㅛㅇ되는 클래스나 ID가 있는지 확인
                - 불풀요하게 많은 class 선택자를 사용할 필요는 없음
                DOM 트리 상단에 같은 스타일을 공유하는 클래스나 ID가 있다면, 함께사용

    4. ID 선택자
        : ID 선택자는 마침표 대신 #을 사용하며, class 선택자와 매우 유사
            >> 클래스 선택자와 ID 선택자 중 선택할 때 생각할 것들
                1. 한 페이지 내에서 여러번 반복될 필요가 있는 스타일은 클래스 선택자를
                한번 유일하게 적용될 스타일은 ID 선택자를 사용하는 것이 좋다 이유는 다음과 같다
                    - class 속성은 어떤 "분류" 안에 포함된 요소의 특성을 정의하는 데 사용
                    - ID 속성은 어떤 요소에 대해 유일한 특성을 정의
                
                2. class 선택자는 글자색이나 글자 굵기 등 다른 곳에도 적용할 수 있는 스타일을 지정,
                ID 선택자는 웹 문서 안ㅇ서 요소의 배치 방법을 지정할 때 자주 사용

                3. class 소성은 속성값을 두 개 이상 가질 수 있음
                -> 한 태그 내에서도 여러 종류의 스타일 규칙을 적용할 수 있음

                4. ID 선택자의 우선순위가 class 선택자의 우선순위 보다 높음
                -> 우선시 되어야 할 스타일을 ID 선택자를 사용하여 정의
    
    4. 복합 선택자
        : 두 개 이상의 선택자 요사 모인 선택자.
            - 하위 선택자와 자식 선택자

            /* CSS */
            /* 하위 선택자 */
            section ul { border: 1px dotted black; }

            /* 자식 선택자 */
            section>ul { border: 1px dotted black; }

            태그들이 포함 관계를 가질 때 포함하는 요소를 부모 요소, 포함되는 요소를 자식 요소라고 한다.
            하위 선택자는 부모 요소에 포함된 모든 하위 요소에 스타일을 적용한다.
            그러나 자식 선택자는 부모 바로 아래 자식 요소에만 적용한다.

            - 인접 형제 선택자와 일반 형제 선택자
            /* CSS */
            /* 인접 형제 선택자 */
            h1+ul { background: yellowgreen; color: darkgreen; }

            /* 일반 형제 선택자 */
            h1~ul { background: darkgreen; color: yellowgreen; }

            같은 부모 요소를 가지는 요소들을 현제 관계라고 부른다. 이때 먼저 나오는 요소를 '형 요소' 나중에 나오는 요소를 '동생 요소'라고 한다

            인접 형제 선택자는 형제 중 첫번째 동생 요소가 조건을 충족 시킬 때에만 스타일을 적용
            일반 형제 선택자는 조건을 충족하는 모든 동생 요소에 스타일을 적용
    
    5. 속성 선택자
        : 속성값의 조건에 따라 다양한 스타일을 지정할 수 있는 스타일 지정방식
        1. E[attr] : 'attr' 속성이 포함된 요소 E 를 선택한다.
        2. E[attr="val"] : 'attr' 속성의 값이 정확하게 'val'과 일치하는 요소 E를 선택한다
        3. E[attr~="val"] : 'attr' 속성의 값에 'val'이 포함되는 요소를 선택한다
        4. E[attr^="val"] : 'attr' 속성의 값이 'val'으로 시작하는 요소를 선택한다.
        5. E[attr$="val"] : 'attr' 속성의 값이 'val'으로 끝나는 요소를 선택한다.
        6. E[attr*="val"] : 'attr' 속성의 값에 'val'이 포함되는 요소를 선택한다.
        7. E[attr|="val"]	'attr' 속성의 값이 정확하게 'val' 이거나
        'val-' 으로 시작되는 요소 E를 선택한다.
```

# 웹 안전 서체
```
    우리가 서체를 선언할 때 그 서체가 방문자의 PC에는 없을 수 있다.
    만약 서체가 우리가 의도한대로 나오지 않는다면 우리가 원하던 디자인과는
    많이 차이가 생길것이다. 때문에 서체를 선언할 때는 대부분의 사람들이
    가지고 있을만한 서체를 선언하는 것이 좋다. 이러한 서체들을 '웹 안전 서체'
    라고 부른다.
```
> font-size : 글씨의 크기를 나타내는 속성 값이다
> font-style
-> 글씨의 스타일을 주는 속성이다.
    가능 속성값
    - normal : 기본
    - italic : 이탤릭체
    - oblique : 기울임꼴
    - inherit : 상위 요소의 font-style을 물려받음

> font-weight : 글씨의 굵기

> font-variant : 대소문자에 대한 스타일
    - nomal : 기본적인 형태
    - small-caps : 소문자를 작은 대문자의 형태
    - inherit : 상위요소의 값을 상속

> line-height : 줄 높이
> font : 글자 관련 속성을 font 하나로 묶어서 사용 가능
