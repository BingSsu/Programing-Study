# HTML
```
HTML은 프로그래밍 언어는 아니고 우리가 보는 웹페이지가 어떻게 구조화되어 있는지 브라우저로하여금 알 수 있도록 하는 마크업 언어이다.

HTML은 elements로 구성되어 있으며 각 컨텐츠의 부분들은 감싸고 마크업 한다. tags는 웹상의 다른 페이지로 이동하게 하는 하이퍼링크 내용들은 생성하거나, 단어를 강조하는 등의 역할을 한다.
```
## Head
```
    HTML은 head section과 body section으로 나누는데 head에는 이 html 문서가  
    어떤 문서인지 명시하는 정보들이 들어간다.

    다음과 같은 태그들이 들어간다
    1. <title> : 웹 페이지의 제목을 나타내는 태그
    2. <meta> : 웹 페이지의 보이지 않는 정보를 제공하는 태그
        - description, keywords, author, subject, classification 등의 정보를 표기할 수 있으며 검색 엔진은 이런 정보를 적극적으로 활용한다.

            
```

## Body
```
    문서의 몸통을 나나태는 태그이다.
    <head>태그와 대조적이며, 브라우저 화면에 보이는것들이 주로 들어간다

    다음과 같은 태그들이 들어간다
    1. <br> : 줄바꿈 태그
    2. <p> : 문단을 만들때 쓰는 태그
    3. <b> : 글자를 굵게 표시하는 태그
        -> 최신 표준은 <string></string>을 권장한다
    4. <i> : 글자 기울임 태그
    5. <h#> : 센션, 문단의 제목을 나타내는 태그
        -> # 자리에 들어가는 숫자가 작을 수록 글자크기가 커진다.
    6. <a> : 하이퍼링크를 걸어주는 태그
        -> href : 클릭시 이동할 링크
        -> target 링크를 여는 방법
            - _self : 현재 페이지에 열기
            - _blank : 새탭으로 열기
            - _parent : 부모페이지로 열기
            - _top : 최상위 페이지로 열기
            - 프레임이름 : 직접 프레임 이름을 명시햇 사용 가능
    7. <img> : 이미지를 삽입하는 태그 src 속성을 통해 이미지 경로를 지정합니다.
        - src : 이미지의 경로
        - width : 이미지 가로크기
        - height : 이미지 세로크기
    8. <table> : 표를 만드는 태그
        - <tr> : 표의 열
        - <td> : 표의 행
        - <thead> : 제목 영역을 나타내는 태그(<table> 하위, <tr> 상위)
        - <tbody> : 본문 영역을 나타내는 태그 (thead와 같은 위치)
        - <th> : 제목 셀을 나타내는 태그 td 대신 사용
    9. <div> : 레이아웃을 나누는데 주로 쓰는 태그(CSS와 연동)
    10. <span> : CSS와 함께 쓰임, 글자에 색 블록을 넣을 수 있음(inline)
    11. <ol> : 번호를 메겨 순서가 있는 목록을 만드는 태그
    12. <ul> : 번호없이 모양으로 목록을 만드는 태그
    13. <li> : <ul></li> 혹은 <ol></ol>와 태그 내부에 들어가는 목록을 만드는 
    태그
    14. <from> : 웹페이지에서 입력양식을 의미하는 태그
        속성
        - name : 폼의 이름
        - action : 폼 데이터가 전송되는 백엔드 url
        - method : 폼 전송 방식(GET/POST)

        <input> : 입력을 받는 태그
            type
            - text : 일반 문자
            - password : 비밀번호
            - button : 버튼
            - submit : 양식 제출용 버튼
            - reset : 양식 초기화용 버튼
            - radio : 하나만 선택할 수 있는 컴포넌트
            - checkbox : 다수를 선택할 수 있는 컴포넌트
            - file : 파일 업로드
            - hidden : 사용자에게 보이지 않는 숨은 요소
```
